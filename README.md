# 🔧 Alan - Servicios Técnicos y Mecánicos

Bienvenido a **Alan**, una plataforma integral de servicios técnicos e mecánicos con acceso rápido mediante códigos QR.

## ✨ Características

### 🎯 Características Principales

- **Acceso rápido con QR**: Escanea códigos QR para acceder al sistema de reservas y catálogo de servicios
- **Servicios Técnicos**: Reparación de computadoras, instalación de software, mantenimiento, recuperación de datos, configuración de redes
- **Servicios Mecánicos**: Cambio de aceite, reparación de frenos, alineación, cambio de batería, reparación de motor
- **Personal Profesional**: Equipo técnico y mecánico altamente capacitado con años de experiencia
- **Precios Competitivos**: Tarifas razonables y transparentes para todos los servicios
- **Diseño Responsivo**: Funciona perfectamente en dispositivos móviles y de escritorio

## 📱 Cómo Usar

### Para Agendar una Cita
1. Abre el navegador web
2. Navega a la sección "Acceso QR"
3. Escanea el código QR con tu smartphone
4. Sigue los pasos para agendar tu cita

### Para Ver el Catálogo de Servicios
1. Escanea el segundo código QR
2. Consulta todos nuestros servicios disponibles
3. Verifica precios y descripciones
